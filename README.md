# Netflix Clone

a react netflix clone using [The Movie Database](https://www.themoviedb.org/) api
made after watching the react master class course of [Nomad Coders](https://nomadcoders.co/).

## Used Libraries

- `react-helmet`
- `react-hook-form`
- `react-player`
- `react-query`
- `react-router-dom`
- `recoil`
- `styled-components`
- `framer-motion`
